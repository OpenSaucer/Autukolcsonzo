# Autókölcsönző Rendszer (OOP Projekt)

Ez a projekt egy autókölcsönző működését szimulálja Python nyelven.

## Alkalmazott OOP technológiák és minták:
- **Absztrakt osztályok (`ABC`):** Az `Auto` alaposztály definiálására.
- **Öröklődés és Polimorfizmus:** `Szemelyauto` és `Teherauto` osztályok egyedi bérletidíj-számítással.
- **Adatvédelem (Encapsulation):** Non-public attribútumok használata getter/setter (`@property`) eléréssel.
- **Validáció és Hibakezelés:** Dátumformátumok, ütközések és múltbéli foglalások szűrése `try-except` blokkokkal.

## Indítás
A program futtatásához csak indítsa el az `autokolcsonzo.py` fájlt.
